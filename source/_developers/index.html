Safety Server

EnMirynet의 서버는 경기도 오산시에 위치해있으며, Replit과 병행하여 운영 중입니다.

  • Back-End : NodeJS (Express)
  • 모든 백엔드 서버 SHA2 암호화
  • 모든 내부 API 코드 난독화 (Base64 or RC4)

Safety Website

EnMirynet의 모든 웹사이트는 SSL 인증서가 설치되어있습니다.

  • Front-End : (HTML, CSS, CommonJS), BootStrap, jQuery
  • TLS WWW 서버, TLS WWW 클라이언트 인증
  • SSL : R3 인증서 (Let's Encrypt)

Safety Domain

EnMirynet의 모든 사이트는 dotname에서 구매한 정품 국제 도메인을 사용합니다

  • WHOIS 도메인 정보 등록 완료
  • 중요 도메인 서비스 등록 완료
  • 도메인 소유권 : EnMirynet 대표 (권도혁)
